HashBot

Hashbot is a commandline tool that generates data for HashBox.

To get started you can either:

$ npm install hashbot

or 

$ git clone https://github.com/draco2003/node-hashbot/

What does it do?

Via the config file, point HashBot at a directory of files and it will generate a hash for each file, and send the filename and hash to HashBox for storage and/or verification.

Configuration Options:

Example Config:

{
  "hashbox": {
    "host" : "httpp://127.0.0.1:9999"
    "create" : "/api/hash_create",
    "verify" : "/api/hash_verify"
  },
  "hash": {
    "threads": 1
  },
  "scandir": {
    "dir": "./node_modules",
    "recursive": true,
    "filter": "/.js/"
  }
}

Running HashBot

Its as simple as:

cd ./node-hashbot
node .
